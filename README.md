# 使用步骤

 1、引入手淘的开源库Flexible。

 2、1rem=75px  目前Flexible会将视觉稿分成100份   750/100=75

 3、安装自动换算插件 CSSREM （自行百度谷歌安装）

 4、开始撸吧


 友情提醒  640/750 设计图  都不用除以2
